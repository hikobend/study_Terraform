# 学習ステップ

1. 構成図を作成する

2. コード一文ごとの内容をメモする

3. resourceごとに表を作成する

4. 1〜3を繰り返す

## terraform初期設定
・terraform initをして初期化

・Terraform用IAMユーザーを作成

・ターミナルにIAMユーザーを登録
